Overview

LoanManager Pro is a sophisticated desktop/web application designed for financial professionals to manage and analyze loans with precision. Built with Python and PyQt5, it combines powerful backend calculations with an intuitive user interface to provide comprehensive loan management capabilities.

Key Features

Loan Calculations

  • French and Italian amortization methods
  • Multiple payment frequencies (monthly, quarterly, semi-annual, annual)
  • Flexible downpayment options
  • TAEG (APR) calculations

Technical Specifications

Backend (loan_analyst.py)

The backend is built with Python and provides robust financial calculations and software integrations including:

  • numpy-financial for core financial computations
  • pandas for data manipulation and analysis
  • matplotlib and seaborn for data visualization
  • PostgreSQL for database operations
  • Advanced TAEG calculations using numerical methods
  • AI chatbot agent for automating tasks

Frontend (loan_app.py)

The user interface is developed using PyQt5 and features:

  • Modern and responsive design
  • Command pattern for undo/redo operations
  • Multiple specialized dialogs for different operations
  • Real-time data validation and error handling
  • Interactive data visualization tools

Architecture Overview

Core Components

  • DbManager Class The database class managing all database operations
  • Loan Class: The central class managing all loan-related calculations and data
  • LoanApp Class: The main application window handling user interactions
  • Specialized Dialogs: Modular components for specific operations
  • Database Integration: Persistent storage system for loan data
  • AI agent integration: Integration of an AI agent chatbot for automating tasks

Key Design Patterns

  • Command Pattern: Used for implementing undo/redo functionality
  • Observer Pattern: For real-time updates of UI elements
  • Factory Pattern: In loan creation and management
  • Singleton Pattern: For database connection management
